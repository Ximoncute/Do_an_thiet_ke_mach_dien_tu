## 📋 Mô tả đồ án

**Đồ án Thiết kế Mạch Điện Tử** là dự án cuối kỳ / đồ án tốt nghiệp thuộc chuyên ngành **Điện tử - Viễn thông / Kỹ thuật Điện tử**.

Đồ án tập trung vào việc thiết kế, mô phỏng, lập trình và hiện thực hóa một hệ thống nhúng hoàn chỉnh, bao gồm:
- Thiết kế sơ đồ nguyên lý (Schematic)
- Thiết kế mạch in (PCB Layout)
- Lập trình vi điều khiển
- Giao tiếp cảm biến, actuator và các module ngoại vi
- Hiển thị và truyền dữ liệu (LCD, OLED, UART, I2C, SPI, MQTT, v.v.)

### Tên đề tài (bạn có thể thay đổi)
> **Thiết kế và chế tạo hệ thống [Tên hệ thống cụ thể]** sử dụng vi điều khiển STM32 / Arduino / ESP32 / PIC...

*(Ví dụ: Hệ thống giám sát và điều khiển môi trường nhà kính, Hệ thống cân điện tử thông minh, Robot tránh vật cản, Thiết bị đo huyết áp tự động, Hệ thống chiếu sáng thông minh, ...)*

## ✨ Tính năng chính

- [ ] Thiết kế mạch nguồn ổn định (3.3V, 5V, 12V...)
- [ ] Giao tiếp với các cảm biến (Nhiệt độ, độ ẩm, ánh sáng, khoảng cách, khí gas, trọng lượng...)
- [ ] Điều khiển các thiết bị ngoại vi (Động cơ, Relay, Servo, LED, Buzzer, LCD/OLED...)
- [ ] Giao tiếp không dây (WiFi, Bluetooth, LoRa, Zigbee...)
- [ ] Hiển thị dữ liệu thời gian thực trên màn hình
- [ ] Lưu trữ dữ liệu (SD Card, EEPROM, Flash)
- [ ] Giao diện người dùng (nút bấm, rotary encoder, touch screen...)
- [ ] Truyền dữ liệu lên cloud (ThingSpeak, Blynk, MQTT, Firebase...)
- [ ] Thiết kế PCB 2 lớp / 4 lớp chuyên nghiệp

## 🚀 Cách sử dụng / Hướng dẫn triển khai

1. Mở file sơ đồ nguyên lý bằng **Altium Designer** hoặc **KiCad**
2. Mở project lập trình bằng phần mềm tương ứng (STM32CubeIDE / Arduino IDE...)
3. Compile và nạp code vào vi điều khiển
4. Kiểm tra và hiệu chỉnh phần cứng
5. Thực hiện đo lường, kiểm tra tính năng
