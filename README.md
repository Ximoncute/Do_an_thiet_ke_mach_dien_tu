## 📋 Mô tả đồ án

**Đồ án Thiết kế Mạch Điện Tử** là dự án cuối kỳ / đồ án tốt nghiệp thuộc chuyên ngành **Điện tử - Viễn thông / Kỹ thuật Điện tử**.

Đồ án tập trung vào việc thiết kế, mô phỏng, lập trình và hiện thực hóa một hệ thống nhúng hoàn chỉnh, bao gồm:
- Thiết kế sơ đồ nguyên lý (Schematic)
- Thiết kế mạch in (PCB Layout)
- Lập trình vi điều khiển
- Giao tiếp cảm biến, actuator và các module ngoại vi
- Hiển thị và truyền dữ liệu (LCD, OLED, UART, I2C, SPI, MQTT, v.v.)

# Đồ án: Thiết Kế Mạch Điện Tử

## Giới thiệu
Đây là repository lưu trữ toàn bộ tài liệu, mã nguồn, và các file thiết kế cho đồ án môn học **Thiết kế mạch điện tử**.

Mục tiêu của dự án là:
- Thiết kế và mô phỏng một hệ thống mạch điện tử hoàn chỉnh.
- Phân tích nguyên lý hoạt động của mạch.
- Kiểm tra hoạt động thông qua mô phỏng và/hoặc thực nghiệm.
- Tối ưu hiệu năng và độ ổn định của hệ thống.

Repository này được sử dụng để:
- Lưu trữ source code
- Lưu trữ file thiết kế
- Theo dõi quá trình phát triển dự án

GitHub giúp quản lý source code và theo dõi thay đổi trong dự án một cách hiệu quả. :contentReference[oaicite:1]{index=1}

# Nội dung dự án

Dự án bao gồm các phần chính:

## 1. Phân tích yêu cầu hệ thống
- Xác định chức năng của mạch
- Xác định các thông số kỹ thuật
- Chọn linh kiện phù hợp

## 2. Thiết kế mạch
- Sơ đồ nguyên lý
- Tính toán linh kiện
- Thiết kế PCB (nếu có)

## 3. Mô phỏng
Các phần mềm có thể sử dụng:
- Proteus
- Multisim
- LTspice
- Altium Designer

Mục đích:
- Kiểm tra hoạt động mạch
- Phát hiện lỗi thiết kế
- Tối ưu hệ thống

## 4. Thực nghiệm (nếu có)
- Lắp ráp mạch
- Kiểm tra tín hiệu
- Đo điện áp / dòng điện
- So sánh với mô phỏng

---

## 5 Mở file mô phỏng
Mở bằng:
- Proteus
hoặc
- Multisim

## 6 Chạy mô phỏng
- Run simulation
- Quan sát tín hiệu đầu ra
